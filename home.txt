На 8 балів:
1. Змінити структуру даних:
Додайте у файл chat.json поле timestamp до кожного повідомлення,
яке буде відображати час створення повідомлення. Ось приклад нового формату:
{
  "users": [
    { "id": 1, "name": "John Doe" },
    { "id": 2, "name": "Jane Smith" }
  ],
  "messages": [
    { "id": 1, "authorId": 1, "body": "Hello!", "timestamp": "2023-09-12T12:34:56Z" },
    { "id": 2, "authorId": 2, "body": "Hi there!", "timestamp": "2023-09-12T12:35:10Z" }
  ]
}
2. Відображення часу повідомлення:
Відредагувати компонент Chat2, щоб поряд з автором і текстом повідомлення відображався
час надсилання повідомлення у зручному для користувача форматі
(наприклад, 12 вересня 2023, 12:34).

На 12 балів:
Попередні завдання + 

3. Обробка помилок:
Зробити так, щоб якщо під час завантаження даних виникла помилка
(наприклад, JSON-файл містить помилку),
в інтерфейсі з'являлось повідомлення про помилку.
Це повідомлення має виводити текст з state.error.

4. Фільтрація повідомлень:
Додати до компонента поле вводу, яке дозволить фільтрувати повідомлення за автором.
При введенні тексту в поле мають відображатися тільки ті повідомлення,
де ім'я автора містить цей текст.

5.* Додати можливість сортувати повідомлення за датою (в порядку зростання або спадання).
Це можна зробити за допомогою випадаючого списку для перемикання порядку сортування.
